########################## SYSTEME DE GESTION DE BIBLIOTHEQUE PHP & MySQL  ################################

--Description : système de gesion de bibliothèque interactif

1- Gestion des livres :
	*Admin : connectez-vous à l'interface admin et faite l'ajout de livre

2- Gestion des emprunts 
	*Utilisateurs :
		- créez votre compte utilisateur
		- faite une demande d'emprunt de livre (clicez sur le bt "borrow")
		- les livres emprunter apparraissent sur votre profil
	
	*Admin :
		- validez les demandes d'emprunt via votre interface
		- faite le retour des livres empruntés via votre interface

3- Authentification :
	*Admin : email : admin@library.admin
			mdp : adminLibrary
	

	*utilisateur par défaut : 
			email : users@email.mail
			mdp : azerty123


	* créez un compte utilisateur
			cliquez sur "create account"
			renseignez votre email et mdp
			completez vos informations sur votre profil

4- Recherche et filtrage
	*faites un filtrage par livres disponibles ou empruntés (button available)
	*faites une recherche par "nom", "genre", "auteur"
